# Ecommerce Admin Dashboard

## Setup Instructions

### Prerequisites
- Node.js (v16+)
- npm or yarn

### Install Dependencies
```bash
npm install
# or
yarn install
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
